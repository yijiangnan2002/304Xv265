Detailed Description

Sent in response to requesting the advertising sets' information using the ConnectionDmBleExtAdvSetsInfoReq() function.

Note: Adv set 0 is for the legacy advertising/connect API usage and will always be registered and may show as advertising if enabled by the legacy API.
This message is sent to the task specified in the above function.

This is a BT5.0+ message.
